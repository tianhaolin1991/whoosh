# ohos.image.interop（图片处理互操作）

本模块提供 PixelMap 进行互操作的能力。

```cangjie
import kit.ImageKit.*
```
## Interfaces

### JSSystemObjectInteropType

interface JSSystemObjectInteropType
```cangjie
public interface JSSystemObjectInteropType { static func fromJSValue(context: JSContext, input: JSValue): PixelMap func toJSValue(context: JSContext): JSValue }
```
功能： 提供 PixelMap 进行互操作的能力。

#### func toJSValue(context: JSContext): JSValue

```cangjie
func toJSValue(context: JSContext): JSValue
```
