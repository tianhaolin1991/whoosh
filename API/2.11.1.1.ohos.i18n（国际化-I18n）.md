# ohos.i18n（国际化-I18n）

本模块提供系统相关的或者增强的国际化能力，包括区域管理、电话号码处理、日历等，相关接口为ECMA 402标准中未定义的补充接口。Intl模块提供了ECMA 402标准定义的基础国际化接口，与本模块共同使用可提供完整地国际化支持能力。

```cangjie
import kit.LocalizationKit.*
```
## Functions

### public func getCalendar(locale: String, calendarType!: ?String = None): Calendar

```cangjie
public func getCalendar(locale: String, calendarType!: ?String = None): Calendar
```
功能： 获取日历对象。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| String | 是 | 表示区域信息的字符串，由语言、脚本、国家或地区组成，例如zh-Hans-CN。 | locale | - |
| ?String | 否 | 命名参数。 合法的日历类型，目前合法的类型有buddhist, chinese, coptic, ethiopic, hebrew, gregory, indian, islamic_civil, islamic_tbla, islamic_umalqura, japanese, persian。取默认值None时是区域默认的日历类型。 | calendarType | None |

**Returns:**

| type | description |
| --- | --- |
| Calendar | 日历对象。 |

**Example:**

```cangjie
// index.cj

import kit.LocalizationKit.*

let calendar = getCalendar("zh-Hans",calendarType: "chinese") // 获取中国农历日历对象
```
