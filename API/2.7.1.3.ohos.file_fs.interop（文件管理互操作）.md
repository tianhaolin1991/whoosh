# ohos.file_fs.interop（文件管理互操作）

本模块提供File、RandomAccessFile以及Stream进行互操作的能力。

```cangjie
import kit.CoreFileKit.*
```
## Interfaces

### JSSystemObjectInteropType

interface JSSystemObjectInteropType
```cangjie
public interface JSSystemObjectInteropType<T> { static func fromJSValue(context: JSContext, input: JSValue): T func toJSValue(context: JSContext): JSValue }
```
功能： JS系统对象专用的拓展接口，以实现与 JSValue 的互转。

#### func toJSValue(context: JSContext): JSValue

```cangjie
func toJSValue(context: JSContext): JSValue
```
功能： 将仓颉对象转换成 JSValue 。

**Parameters:**

| type | description | name | required |
| --- | --- | --- | --- |
| JSContext | ArkTS互操作上下文。 | context | 是 |

**Returns:**

| type | description |
| --- | --- |
| JSValue | ArkTS统一类型。 |



## Classes

### Stream

class Stream
```cangjie
extend Stream <: JSSystemObjectInteropType<Stream> {}
```
功能： 文件流，在调用Stream的方法前，需要先通过 FileFs.createStream 方法或者 FileFs.fdopenStream 来构建一个Stream实例。


### File

class File
```cangjie
extend File <: JSSystemObjectInteropType<File> {}
```
功能： 由open接口打开的File对象。


### RandomAccessFile

class RandomAccessFile
```cangjie
extend RandomAccessFile <: JSSystemObjectInteropType<RandomAccessFile> {}
```
功能： 随机读写文件流，在调用RandomAccessFile的方法前，需要先通过 createRandomAccessFile 方法来构建一个RandomAccessFile实例。
