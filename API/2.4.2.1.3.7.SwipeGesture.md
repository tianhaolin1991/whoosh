# SwipeGesture

用于触发滑动事件，滑动速度大于100vp/s时可识别成功。

## Component Constructors

### public init(fingers!: Int32 = 1, direction!: SwipeDirection = SwipeDirection.All, speed!: Float64 = 100.0)

```cangjie
public init(fingers!: Int32 = 1, direction!: SwipeDirection = SwipeDirection.All, speed!: Float64 = 100.0)
```
功能： 创建一个滑动手势。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Int32 | 否 | 命名参数。 触发滑动的最少手指数，默认为1，最小为1指，最大为10指。 | fingers | 1 |
| SwipeDirection | 否 | 命名参数。 触发滑动手势的滑动方向。 | direction | SwipeDirection.All |
| Float64 | 否 | 命名参数。 识别滑动的最小速度。 说明： 当滑动速度的值小于等于0时，会被转化为默认值。 | speed | 100.0 |


## Functions

### public func onAction(callback: (GestureEvent) -> Unit): This

```cangjie
public func onAction(callback: (GestureEvent) -> Unit): This
```
功能： Swipe手势识别成功触发该事件。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ( GestureEvent )->Unit | 是 | 回调函数，Swipe手势识别成功触发。 | callback | - |


## Enums

### SwipeDirection

enum SwipeDirection
```cangjie
public enum SwipeDirection { | Horizontal | Vertical | All | None }
```
功能： 触发滑动手势的滑动方向。

#### All

```cangjie
All
```
功能： 所有方向。

#### Horizontal

```cangjie
Horizontal
```
功能： 水平方向，手指滑动方向与x轴夹角小于45度时触发。

#### None

```cangjie
None
```
功能： 任何方向都不可触发拖动手势事件。

#### Vertical

```cangjie
Vertical
```
功能： 竖直方向，手指滑动方向与y轴夹角小于45度时触发。



## Examples

```cangjie
package ohos_app_cangjie_entry

import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 @State var rotateAngle: Float64 = 0.0
 @State var speed: Float64 = 1.0

 func build() {
 Column() {
 Column() {
 Text("SwipeGesture speed\n" + this.speed.toString())
 Text("SwipeGesture angle\n" + this.rotateAngle.toString())
 }
 .height(200)
 .width(300)
 .padding(20)
 .border(width: 3.vp)
 .margin(80)
 .rotate(Float32(this.rotateAngle))
 .gesture(
 SwipeGesture(direction: SwipeDirection.Vertical)
 .onAction({ event: GestureEvent =>
 this.speed = event.speed
 this.rotateAngle = event.angle
 })
 )
 }
 }
}
```
