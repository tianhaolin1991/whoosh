# ohos.base（公共回调信息）

本模块定义了接口调用时出现的公共异常信息。

```cangjie
import ohos.base.*
```

### AsyncError

class AsyncError
```cangjie
public class AsyncError { public AsyncError( public let code: Int32 ) }
```
功能： 异步操作中的错误码。

```cangjie

```
功能： 错误码。
