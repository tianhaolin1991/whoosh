# ohos.telephony_call（拨打电话）

该模块提供呼叫管理功能，包括拨打电话、跳转到拨号界面、获取通话状态、格式化电话号码等。

```cangjie
import kit.TelephonyKit.*
```
## Classes

### TelephonyCall

class TelephonyCall
```cangjie
public class TelephonyCall {}
```
功能： 拨打电话类，提供呼叫管理功能，包括拨打电话、跳转到拨号界面、获取通话状态、格式化电话号码等接口。


### NumberFormatOptions

class NumberFormatOptions
```cangjie
public class NumberFormatOptions { public let countryCode: String public init(countryCode: String) }
```
功能： 格式化号码的可选参数。

#### let countryCode

```cangjie
public let countryCode: String
```
功能： 国家码，支持所有国家的国家码，如：CN（中国）。默认为：CN。
Type: String; Readable: true; Writable: false



## Enums

### CallState

enum CallState
```cangjie
public enum CallState { | CALL_STATE_UNKNOWN | CALL_STATE_IDLE | CALL_STATE_RINGING | CALL_STATE_OFFHOOK | CALL_STATE_ANSWERED | ... }
```
功能： 通话状态码。

#### CALL_STATE_ANSWERED

```cangjie
CALL_STATE_ANSWERED
```
功能： 表示来电已经接听。

#### CALL_STATE_IDLE

```cangjie
CALL_STATE_IDLE
```
功能： 表示没有正在进行的呼叫。

#### CALL_STATE_OFFHOOK

```cangjie
CALL_STATE_OFFHOOK
```
功能： 表示至少有一个呼叫处于拨号、通话中或呼叫保持状态，并且没有新的来电振铃或等待。

#### CALL_STATE_RINGING

```cangjie
CALL_STATE_RINGING
```
功能： 表示来电正在振铃或等待。

#### CALL_STATE_UNKNOWN

```cangjie
CALL_STATE_UNKNOWN
```
功能： 无效状态，当获取呼叫状态失败时返回。
