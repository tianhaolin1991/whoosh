# ohos.displaySync（可变帧率）

可变帧率支持让开发者以指定帧率来运行UI业务，一般用于开发者自绘制UI，并且对于帧率有特定诉求的场景。

```cangjie
import kit.ArkGraphics2D.*
```
## Classes

### DisplaySync

class DisplaySync
```cangjie
public class DisplaySync {}
```
功能： 帧率和回调函数设置实例。用于帧率设置和回调函数的注册，以及启动和停止回调函数的调用。


### IntervalInfo

class IntervalInfo
```cangjie
public class IntervalInfo { public IntervalInfo ( public var timestamp: Int64, public var targetTimestamp: Int64 ) }
```
功能： 开发者可以从订阅函数中获取帧绘制的时间戳信息，包含当前帧到达的时间timestamp和下一帧预期到达的时间targetTimestamp。

#### var targetTimestamp

```cangjie
public var targetTimestamp: Int64
```
功能： 下一帧预期到达的时间（单位：纳秒）。
Type: Int64; Readable: true; Writable: true

#### var timestamp

```cangjie
public var timestamp: Int64
```
功能： 当前帧到达的时间（单位：纳秒）。
Type: Int64; Readable: true; Writable: true
