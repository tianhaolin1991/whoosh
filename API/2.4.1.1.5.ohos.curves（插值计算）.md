# ohos.curves（插值计算）

本模块提供设置动画插值曲线功能，用于构造阶梯曲线对象、构造三阶贝塞尔曲线对象和构造弹簧曲线对象。

```cangjie
import kit.UIKit.*
```
## Classes

### Curves

class Curves
```cangjie
public class Curves {}
```
功能： 动画插值曲线类型。


### ICurve

class ICurve
```cangjie
public class ICurve {}
```
功能： 曲线对象类型。



## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

let animateOpt1 = AnimateParam(
 duration: 1200,
 curve: Curve.EaseOut
)

@Entry
@Component
class EntryView {
 @State var widthSize: Float64 = 200.0
 @State var heightSize: Float64 = 200.0
 func build() {
 Column {
 Column()
 .margin(top: 100)
 .width(this.widthSize)
 .height(this.heightSize)
 .backgroundColor(Color.RED)
 .onClick(
 {
 evt =>
 let curve = Curves.cubicBezierCurve(0.25, 0.1, 0.25, 1.0)
 this.widthSize = Float64(curve.interpolate(0.5)) * this.widthSize
 this.heightSize = Float64(curve.interpolate(0.5)) * this.heightSize
 }
 )
 .animationStart(animateOpt1)
 }
 }
}
```
