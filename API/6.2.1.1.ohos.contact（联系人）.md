| type | description |

- **BusinessException**: BusinessException：对应错误码的详细介绍请参见 通用错误码 。

| code | description |
| --- | --- |
| 201 | Permission denied. |
| 401 | Parameter error. |


**Example:**

```cangjie
// index.cj

import ohos.base.*
import kit.ContactsKit.*
```

### public func queryContact(context: Context, key: String, holder!: Holder = Holder.EMPTY, attrs!: ContactAttributes = ContactAttributes.ALL ): Contact

```cangjie
public func queryContact(context: Context, key: String, holder!: Holder = Holder.EMPTY, attrs!: ContactAttributes = ContactAttributes.ALL ): Contact
```
功能： 根据key，holder和attrs查询联系人。当不存在该联系人时，抛出401异常。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Context | 是 | 应用上下文Context。 | context | - |
| String | 是 | 联系人的key值，一个联系人对应一个key。 | key | - |
| Holder | 否 | 命名参数。 创建联系人的应用信息。 | holder | Holder.EMPTY |
| ContactAttributes | 否 | 命名参数。 联系人的属性列表。 | attrs | ContactAttributes.ALL |

**Returns:**

| type | description |
| --- | --- |
| Contact | 成功返回查询的联系人对象；失败抛出异常。 |

**Exceptions:**

- **BusinessException**: BusinessException：对应错误码的详细介绍请参见 通用错误码 。

| code | description |
| --- | --- |
| 201 | Permission denied. |
| 401 | Parameter error. |


**Example:**

```cangjie
// index.cj

import ohos.base.*
import kit.ContactsKit.*

let contactId = 1
let abilityContext = Global.abilityContext // 需获取Context应用上下文，详见本文使用说明
let key = queryKey(abilityContext, contactId)
let attrs = ContactAttributes([Attribute.ATTR_NAME, ATTR_NICKNAME])
let holder = Holder("displayName", 1, bundleName: "com.ohos.contacts")
let contact = queryContact(abilityContext, key, holder: holder, attrs: attrs)
println(contact.id)
```
