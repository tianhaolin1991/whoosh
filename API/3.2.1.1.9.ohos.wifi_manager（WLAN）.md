# ohos.wifi_manager（WLAN）

该模块主要提供WLAN基础功能、P2P（peer-to-peer）功能和WLAN消息通知的相应服务，让应用可以通过WLAN和其他设备互联互通。

```cangjie
import kit.ConnectivityKit.*
```
