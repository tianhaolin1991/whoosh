public func contentFont( size!: Length = 16.vp, weight!: FontWeight = FontWeight.Normal, family!: String = "HarmonyOS Sans", style!: FontStyle = FontStyle.Normal ): This
```
功能： 设置菜单项中内容信息的字体样式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 否 | 命名参数。 设置文本尺寸，Length为Int64、Float64类型时，使用fp单位。不支持百分比设置。 | size | 16.vp |
| FontWeight | 否 | 命名参数。 设置文本的字体粗细。 | weight | FontWeight.Normal |
| String | 否 | 命名参数。 设置文本的字体列表。使用多个字体，使用','进行分割，优先级按顺序生效。例如：'Arial, HarmonyOS Sans'。当前支持'HarmonyOS Sans'字体和 注册自定义字体 。 | family | "HarmonyOS Sans" |
| FontStyle | 否 | 命名参数。 设置文本的字体样式。 | style | FontStyle.Normal |
