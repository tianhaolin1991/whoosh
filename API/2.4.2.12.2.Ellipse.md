# Ellipse

椭圆绘制组件。

## Component Constructors

### public init(width!: Length, height!: Length)

```cangjie
public init(width!: Length, height!: Length)
```
功能： 绘制一个宽度为width，高度为height的椭圆。异常值按照初始值处理。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 命名参数。 宽度，取值范围≥0。 初始值：0。 默认单位：vp。 | width | - |
| Length | 是 | 命名参数。 高度，取值范围≥0。 初始值：0。 默认单位：vp。 | height | - |

### public init()

```cangjie
public init()
```
功能： 绘制一个宽度为0，高度为0的椭圆。需要设置 width 或 height 属性参数不为0才能显示出来。


## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Column() {
 // 绘制一个 150 * 80 的椭圆
 Ellipse(width: 150, height: 80)
 // 绘制一个 150 * 100 、线条为红色的椭圆环
 Ellipse()
 .width(150)
 .height(100)
 .fillOpacity(0)
 .stroke(Color.RED)
 .strokeWidth(3)
 .padding(top: 10)
 }.width(100.percent)
 }
}
```
