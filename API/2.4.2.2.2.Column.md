# Column

沿垂直方向布局的容器。

## Component Constructors

### public init()

```cangjie
public init()
```
功能： 创建一个不包含子组件的Column容器。

### public init(child: () -> Unit)

```cangjie
public init(child: () -> Unit)
```
功能： 创建一个可包含子组件的Column容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ()->Unit | 是 | Column 容器的子组件。 | child | - |

### public init(space: Length, child: () -> Unit)

```cangjie
public init(space: Length, child: () -> Unit)
```
功能： 创建一个纵向布局元素垂直方向间距为space且可以包含子组件的Column容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 纵向布局元素垂直方向间距。 space为负数或者 justifyContent 设置为FlexAlign.SpaceBetween、FlexAlign.SpaceAround、FlexAlign.SpaceEvenly时不生效。 初始值：0 单位：vp 说明： space为大于等于0的数字。 | space | - |
| ()->Unit | 是 | Column 容器的子组件。 | child | - |

### public init(space: Length)

```cangjie
public init(space: Length)
```
功能： 创建一个纵向布局元素垂直方向间距为space且可以包含子组件的Column容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 纵向布局元素垂直方向间距。 space为负数或者 justifyContent 设置为FlexAlign.SpaceBetween、FlexAlign.SpaceAround、FlexAlign.SpaceEvenly时不生效。 初始值：0 单位：vp 说明： 可选值为大于等于0的数字。 | space | - |

| type | required | description | name | default |
| --- | --- | --- | --- | --- |

### public func justifyContent(value: FlexAlign): This

```cangjie
public func justifyContent(value: FlexAlign): This
```
功能： 设置子组件在垂直方向上的对齐格式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| FlexAlign | 是 | 设置子组件在垂直方向上的对齐格式。 初始值：FlexAlign.Start | value | - |
