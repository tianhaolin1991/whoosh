# RotationGesture

用于触发旋转手势事件，触发旋转手势的最少手指为2指，最大为5指，最小改变度数为1度。

## Component Constructors

### public init(fingers!: Int32 = 2, angle!: Float64 = 1.0)

```cangjie
public init(fingers!: Int32 = 2, angle!: Float64 = 1.0)
```
功能： 创建一个旋转手势。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Int32 | 否 | 命名参数。 触发旋转的最少手指数, 最小为2指，最大为5指。 说明： 触发手势手指可以多于fingers数目，但只有先落下的两指参与手势计算。 | fingers | 2 |
| Float64 | 否 | 命名参数。 触发旋转手势的最小改变度数，单位为deg。 说明： 当改变度数的值小于等于0或大于360时，会被转化为默认值。 | angle | 1.0 |

## Examples

```cangjie
package ohos_app_cangjie_entry

import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 @State var scaleValue: Float64 = 1.0
 @State var angle: Float64 = 0.0
 @State var rotateValue: Float64 = 0.0

 func build() {
 Column() {
 Column() {
 Text('RotationGesture angle:' + this.angle.toString())
 }
 .height(200)
 .width(300)
 .padding(20)
 .border(width: 3.vp)
 .margin(80)
 .rotate(Float32(this.angle))
 .gesture(
 RotationGesture()
 .onActionUpdate({ event: GestureEvent =>
 this.angle = this.rotateValue + event.angle
 })
 .onActionEnd({ event: GestureEvent =>
 this.rotateValue = this.angle
 })
 )
 }
 }
}
```
