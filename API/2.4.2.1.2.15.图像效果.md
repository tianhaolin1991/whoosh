# 图像效果

设置组件的模糊、阴影、球面效果以及设置图片的图像效果。

## Functions

### public func blendMode(value: BlendMode, `type`: BlendApplyType): This

```cangjie
public func blendMode(value: BlendMode, `type`: BlendApplyType): This
```
功能： 将当前控件的内容（包含子节点内容）与下方画布（可能为离屏画布）已有内容进行混合。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| BlendMode | 是 | 混合模式。 初始值：BlendMode.NONE。 说明： 混合模式设置为BlendMode.NONE时，blend效果实际为默认的BlendMode.SRC_OVER，且BlendApplyType不生效。 | value | - |
| BlendApplyType | 是 | blendMode实现方式是否离屏。 说明： 1. 设置BlendApplyType.FAST时，不离屏。 2. 设置BlendApplyType.OFFSCREEN时，会创建当前组件大小的离屏画布，再将当前组件（含子组件）的内容绘制到离屏画布上，再用指定的混合模式与下方画布已有内容进行混合。使用该实现方式时，将导致 linearGradientBlur ， backgroundEffect ， brightness 等需要截屏的接口无法截取到正确的画面。 | blendType | - |
