# ohos.component_utils（ComponentUtils）

提供获取组件绘制区域坐标和大小的能力。

```cangjie
import ohos.component_utils.*
```
## Classes

### ComponentUtils

class ComponentUtils
```cangjie
public class ComponentUtils { }
```
功能： 提供获取指定组件绘制区域坐标和大小。


### ComponentInfo

class ComponentInfo
```cangjie
public class ComponentInfo { public ComponentInfo( public let size: Size, public let localOffset: Offset, public let windowOffset: Offset, public let screenOffset: Offset, public let translate: TranslateResult, public let scale: ScaleResult, public let rotate: RotateResult, public let transform: Array<Float32> ) }
```
功能： 组件实例对象的坐标位置和大小等信息。

Type: Offset; Readable: true; Writable: false

#### let size
