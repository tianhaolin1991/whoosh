# ohos.ui_test（UI测试）

ui_test提供模拟UI操作的能力，供开发者在测试场景使用，主要支持如点击、双击、长按、滑动等UI操作能力。

```cangjie
import kit.TestKit.*
```
## Classes

### Driver

```cangjie
public class Driver {}
```

功能： Driver 类为uitest测试框架的总入口，提供控件匹配、查找、按键注入，坐标点击或滑动，截图等能力。
