# CanvasRenderingContext2D

使用RenderingContext在Canvas组件上进行绘制，绘制对象可以是矩形、文本、图片等。

```cangjie
import kit.UIKit.*
```
## Classes

### CanvasRenderingContext2D

class CanvasRenderingContext2D
```cangjie
public class CanvasRenderingContext2D { public init(settings: RenderingContextSettings) public init(settings: RenderingContextSettings, unit: LengthMetricsUnit) }
```
功能： 表示使用RenderingContext在Canvas组件上进行绘制的类型，绘制对象可以是矩形、文本、图片等。


### RenderingContextSettings

class RenderingContextSettings
```cangjie
public class RenderingContextSettings { public var antialias: Bool public init(antialias!: Bool = false) }
```
功能： 用来配置CanvasRenderingContext2D对象的参数，包括是否开启抗锯齿。

#### var antialias

```cangjie
public var antialias: Bool
```
功能： 表明canvas是否开启抗锯齿。
