# RowSplit

将子组件横向布局，并在每个子组件之间插入一根纵向的分割线。

## Component Constructors

### public init(content: () -> Unit)

```cangjie
public init(content: () -> Unit)
```
功能： 创建一个可包含子组件的RowSplit容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ()->Unit | 是 | 声明容器内的子组件。 | content | - |


## Functions

### public func resizeable(value: Bool): This

```cangjie
public func resizeable(value: Bool): This
```
功能： 设置分割线是否可拖拽。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Bool | 是 | 分割线是否可拖拽。 初始值：false。 | value | - |


## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Column() {
 Text("The second line can be dragged")
 .fontSize(9)
 .fontColor(0xCCCCCC)
 .width(90.percent)
 RowSplit() {
 Text("1")
 .width(10.percent)
 .height(100)
 .backgroundColor(0xF5DEB3)
 .textAlign(TextAlign.Center)
 Text("2")
 .width(10.percent)
 .height(100)
 .backgroundColor(0xD2B48C)
 .textAlign(TextAlign.Center)
 Text("3")
 .width(10.percent)
 .height(100)
 .backgroundColor(0xF5DEB3)
 .textAlign(TextAlign.Center)
 Text("4")
 .width(10.percent)
 .height(100)
 .backgroundColor(0xD2B48C)
 .textAlign(TextAlign.Center)
 Text("5")
 .width(10.percent)
 .height(100)
 .backgroundColor(0xF5DEB3)
 .textAlign(TextAlign.Center)
 }
 .resizeable(true) // 可拖动
 .width(90.percent).height(100)
 }
 .width(100.percent)
 .padding(top: 5)
 }
}
```
