# 图形变换

用于对组件进行旋转、平移、缩放、矩阵变换等操作。

## Functions

### public func rotate( x!: Float32 = 0.0, y!: Float32 = 0.0, z!: Float32 = 0.0, angle!: Float32 = 0.0, centerX!: Length = 50.percent, centerY!: Length = 50.percent ): This

```cangjie
public func rotate( x!: Float32 = 0.0, y!: Float32 = 0.0, z!: Float32 = 0.0, angle!: Float32 = 0.0, centerX!: Length = 50.percent, centerY!: Length = 50.percent ): This
```
功能： 设置组件旋转。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Float32 | 否 | 命名参数。 旋转轴向量x坐标。 | x | 0.0 |
| Float32 | 否 | 命名参数。 旋转轴向量y坐标。 | y | 0.0 |
| Float32 | 否 | 命名参数。 旋转轴向量z坐标。 | z | 0.0 |
| Float64 | 否 | 旋转角度。取值为正时相对于旋转轴方向顺时针转动，取值为负时相对于旋转轴方向逆时针转动。 | angel | 0.0 |
| Length | 否 | 命名参数。 变换中心点x轴坐标。表示组件变换中心点（即锚点）的x方向坐标。 | centerX | 50.percent |
| Length | 否 | 命名参数。 变换中心点y轴坐标。表示组件变换中心点（即锚点）的y方向坐标。 | centerY | 50.percent |

## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Column() {
 Text("rotate").width(90.percent).fontColor(0xCCCCCC).padding(15).fontSize(14)
 Row()
 .rotate(
 x: 1.0,
 y: 1.0,
 z: 1.0,
 angle: 300.0,
 centerX: 50.percent,
 centerY: 50.percent
 ) // 组件以矢量(1,1,1)为旋转轴，绕中心点顺时针旋转300度
 .width(100).height(100).backgroundColor(0xAFEEEE)

 Text("translate").width(90.percent).fontColor(0xCCCCCC).padding(10).fontSize(14)
 Row()
 .translate(x: 100, y: 10) // x轴方向平移100，y轴方向平移10
 .width(100).height(100).backgroundColor(0xAFEEEE).margin(bottom: 10)

 Text("scale").width(90.percent).fontColor(0xCCCCCC).padding(15).fontSize(14)
 Row()
 .scale(x: 2.0, y: 0.5) // 高度缩小一倍，宽度放大一倍，z轴在2D下无效果
 .width(100).height(100).backgroundColor(0xAFEEEE)
 }.width(100.percent).margin(top: 5)
 }
}
```
