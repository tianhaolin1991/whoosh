# 点击回弹效果

设置组件点击时回弹效果。

## Functions

### public func clickEffect(level!: ClickEffectLevel = ClickEffectLevel.LIGHT, scale!: Option<Float32> = Option.None): This

```cangjie
public func clickEffect(level!: ClickEffectLevel = ClickEffectLevel.LIGHT, scale!: Option<Float32> = Option.None): This
```
功能： 设置当前组件点击回弹效果。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ClickEffectLevel | 否 | 命名参数。 当前组件点击回弹效果。 | level | ClickEffectLevel.LIGHT |
| Option<Float32> | 否 | 命名参数。 回弹缩放比例，支持在设置ClickEffectLevel的基础上微调回弹缩放比例。 说明： level等于ClickEffectLevel.LIGHT时，初始值：0.90。 level等于ClickEffectLevel.MIDDLE或者ClickEffectLevel.HEAVY时，初始值：0.95。 | scale | Option.None |

}
