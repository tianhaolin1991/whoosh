# Polygon

多边形绘制组件。

## Component Constructors

### public init(width!: Length, height!: Length)

```cangjie
public init(width!: Length, height!: Length)
```
功能： 绘制一个宽度为width，高度为height的多边形。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 命名参数。 宽度，取值范围≥0。 初始值：0。 默认单位：vp。 异常值按照初始值处理。 | width | - |
| Length | 是 | 命名参数。 高度，取值范围≥0。 初始值：0。 默认单位：vp。 异常值按照初始值处理。 | height | - |

### public init()

```cangjie
public init()
```
功能： 绘制一个宽度为0，高度为0的多边形。需要设置 width 或 height 属性参数不为0才能显示出来。

## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Column(10) {
 // 在 100 * 100 的矩形框中绘制一个三角形，起点(0, 0)，经过(50, 100)，终点(100, 0)
 Polygon(width: 100, height: 100).points([(0, 0), (50, 100), (100, 0)]).fill(Color.GREEN)
 // 在 100 * 100 的矩形框中绘制一个四边形，起点(0, 0)，经过(0, 100)和(100, 100)，终点(100, 0)
 Polygon()
 .width(100)
 .height(100)
 .points([(0, 0), (0, 100), (100, 100), (100, 0)])
 .fillOpacity(0)
 .strokeWidth(5)
 .stroke(Color.BLUE)
 // 在 100 * 100 的矩形框中绘制一个五边形，起点(50, 0)，依次经过(0, 50)、(20, 100)和(80, 100)，终点(100, 50)
 Polygon()
 .width(100)
 .height(100)
 .points([(50, 0), (0, 50), (20, 100), (80, 100), (100, 50)])
 .fill(Color.RED)
 .fillOpacity(0.6)
 }.width(100.percent).margin(10)
 }
}
```
