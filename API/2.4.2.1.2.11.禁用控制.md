# 禁用控制

组件是否可交互，可交互状态下响应 点击事件 、 触摸事件 、 拖拽事件 、 按键事件 、 焦点事件 、 鼠标事件 、 悬浮事件 和 手势事件 。

## Functions

### public func enabled(value: Bool): This

```cangjie
public func enabled(value: Bool): This
```
功能： 设置组件是否可交互。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Bool | 是 | 值为true表示组件可交互，响应点击等操作。 值为false表示组件不可交互，不响应点击等操作。 初始值：true。 | value | - |


## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Flex(FlexParams(justifyContent: FlexAlign.SpaceAround)) {
 // 点击没有反应
 Button("disable").enabled(false).backgroundColor(0x317aff).opacity(0.4)
 Button("enable").backgroundColor(0x317aff)
 }
 .width(100.percent)
 .padding(top: 5)
 }
}
```
