# Divider

提供分隔器组件，分隔不同内容块/内容元素。

## Component Constructors

### public init()

```cangjie
public init()
```
功能： 创建一个分隔器组件。


## Functions

### public func color(color: ResourceColor): This

```cangjie
public func color(color: ResourceColor): This
```
功能： 设置分割线的颜色。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ResourceColor | 是 | 分割线颜色。 初始值： 0x33182431 | color | - |

### public func lineCap(style: LineCapStyle): This

```cangjie
public func lineCap(style: LineCapStyle): This
```
功能： 设置当前在容器中的分割线端点样式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| LineCapStyle | 是 | 分割线条的端点样式。 初始值： LineCapStyle.Butt。 | style | - |

### public func strokeWidth(width: Length): This

```cangjie
public func strokeWidth(width: Length): This
```
功能： 设置当前在容器中的分割线宽度。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 分割线宽度。不指定像素单位时，默认单位vp。不支持百分比设置。分割线的宽度不支持百分比设置。优先级低于通用属性 height ，超过通用属性设置大小时，按照通用属性进行裁切。部分设备硬件中存在1像素取整后分割线不显示问题，建议使用2像素。 初始值：1.vp | width | - |

### public func vertical(isVertical: Bool): This

```cangjie
public func vertical(isVertical: Bool): This
```
功能： 设置分割线的方向。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Bool | 是 | 使用水平分割线还是垂直分割线。 false：水平分割线， true：垂直分割线。 初始值：false | isVertical | - |


## Examples

package ohos_app_cangjie_entry

import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build() {
 Column() {
 // 使用横向分割线场景
 Text("Horizontal divider")
 .fontSize(9)
 .fontColor(0xCCCCCC)
 List() {
 ForEach(
 [1, 2, 3],
 {
 item: Int64, idx: Int64 => ListItem() {
 Text("list" + item.toString())
 .width(100.percent)
 .fontSize(14)
 .fontColor(0x182431)
 .textAlign(TextAlign.Start)
 }
 .width(244)
 .height(48)
 }
 )
 }.padding(left: 24, bottom: 8)

 Divider()
 .strokeWidth(8)
 .color(0xF1F3F5)

 [4, 5],
 {

 }
 .width(244)
