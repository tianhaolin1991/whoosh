# Flex

Flex是以弹性方式布局子组件的容器组件，提供更加有效的方式对容器内的子元素进行排列、对齐和分配剩余空间。

## Component Constructors

```cangjie
public init(value: FlexParams, child: () -> Unit)
```

## Classes

### FlexOptions

class FlexOptions
```cangjie
public class FlexOptions { public FlexOptions( public var direction!: FlexDirection = FlexDirection.Row, public var wrap!: FlexWrap = FlexWrap.NoWrap, public var justifyContent!: FlexAlign = FlexAlign.Start, public var alignItems!: ItemAlign = ItemAlign.Start, public var alignContent!: FlexAlign = FlexAlign.Start, public var space!: FlexSpaceOptions = FlexSpaceOptions() ) }
```
功能： 表示Flex子组件的排列对齐方式。

#### var alignContent

```cangjie
public var alignContent: FlexAlign = FlexAlign.Start
```
功能： 交叉轴中有额外的空间时，多行内容的对齐方式。仅在wrap为Wrap或WrapReverse下生效。
Type: FlexAlign; Readable: true; Writable: true

#### var alignItems

```cangjie
public var alignItems: ItemAlign = ItemAlign.Start
```
功能： 所有子组件在Flex容器交叉轴上的对齐格式。
Type: ItemAlign; Readable: true; Writable: true

#### var direction

```cangjie
public var direction: FlexDirection = FlexDirection.Row
```
功能： 子组件在Flex容器上排列的方向，即主轴的方向。
Type: FlexDirection; Readable: true; Writable: true

#### var justifyContent

```cangjie
public var justifyContent: FlexAlign = FlexAlign.Start
```
功能： 所有子组件在Flex容器主轴上的对齐格式。
Type: FlexAlign; Readable: true; Writable: true

#### var space

```cangjie
public var space: FlexSpaceOptions = FlexSpaceOptions()
```
功能： 所有子组件在Flex容器主轴或交叉轴的间距。space为负数、百分比或者justifyContent设置为FlexAlign.SpaceBetween、FlexAlign.SpaceAround、FlexAlign.SpaceEvenly时不生效。
Type: FlexSpaceOptions; Readable: true; Writable: true

#### var wrap

```cangjie
public var wrap: FlexWrap = FlexWrap.NoWrap
```
功能： Flex容器是单行/列还是多行/列排列。
Type: FlexWrap; Readable: true; Writable: true
