# ohos.geo_location_manager（位置服务）

位置服务提供GNSS定位、网络定位（蜂窝基站、WLAN、蓝牙定位技术）、地理编码、逆地理编码、国家码和地理围栏等基本功能。

```cangjie
import kit.LocationKit.*
```
