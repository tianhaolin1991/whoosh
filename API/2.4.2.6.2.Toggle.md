# Toggle

组件提供勾选框样式、状态按钮样式及开关样式。

## Component Constructors

### public init(toggleType: ToggleType, isOn!: Bool = false)

```cangjie

```
功能： 创建一个Toggle类型的对象。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ToggleType | 是 | 开关类型。 初始值：ToggleType.Switch。 | toggleType | - |
| Bool | 否 | 命名参数。 开关是否打开。true：打开，false：关闭。 初始值：false。 | isOn | false |

## Functions

### public func selectedColor(color: ResourceColor): This

```cangjie
public func selectedColor(color: ResourceColor): This
```
功能： 设置组件打开状态的背景颜色。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ResourceColor | 是 | 设置组件打开状态的背景颜色。 初始值: 当ToggleType为Switch时，初始值为@r(sys.color.ohos_id_color_component_activated)。 当ToggleType为Checkbox时，初始值为@r(sys.color.ohos_id_color_component_activated)。 当ToggleType为Button时，初始值为@r(sys.color.ohos_id_color_component_activated)混合@r(sys.color.ohos_id_color_text_highlight_bg)的透明度。 | color | - |

### public func switchPointColor(color: ResourceColor): This

```cangjie
public func switchPointColor(color: ResourceColor): This
```
功能： 设置Switch类型的圆形滑块颜色。仅对type为ToggleType.Switch生效。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ResourceColor | 是 | Switch类型的圆形滑块颜色。初始值：@r(sys.color.ohos_id_color_foreground_contrary) | color | - |

### public func switchStyle(pointRadius!: Float64, unselectedColor!: ResourceColor, pointColor!: ResourceColor, trackBorderRadius!: Float64): This

```cangjie
public func switchStyle(pointRadius!: Float64, unselectedColor!: ResourceColor, pointColor!: ResourceColor, trackBorderRadius!: Float64): This
```
功能： 设置Switch类型的样式。仅对toggleType为Switch生效。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Float64 | 是 | 命名参数。 设置Switch类型的圆形滑块半径。 说明： 设定值小于0时按照默认算法设置，设定值大于等于0时按照设定值设置。 未设定此属性时，圆形滑块半径根据默认算法设置。 默认算法：（组件高度（单位：vp） / 2） - （2.vp * 组件高度（单位：vp） / 20.vp）。 | pointRadius | - |
| ResourceColor | 是 | 命名参数。 设置Switch类型关闭状态的背景颜色。 初始值：0x337F7F7F。 | unselectedColor | - |
| ResourceColor | 是 | 命名参数。 设置Switch类型的圆形滑块颜色。 初始值：@r(sys.color.ohos_id_color_foreground_contrary)。 | pointColor | - |
| Float64 | 是 | 命名参数。 设置Switch类型的滑轨的圆角。 说明： 设定值小于0时按照默认算法设置，设定值大于组件高度一半时按照组件高度一半设置，其他场合按照设定值设置。 未设定此属性时，滑轨圆角根据默认算法设置。 默认算法：组件高度（单位：vp） / 2。 | trackBorderRadius | - |

## Enums

### ToggleType

enum ToggleType
```cangjie
public enum ToggleType { | CheckboxType | SwitchType | ButtonType }
```
功能： 开关组件类型。

#### CheckboxType

```cangjie
CheckboxType
```
功能： 提供单选框样式。 Checkbox默认样式为圆形。 通用属性margin的默认值为：top 14.px, right 14.px, bottom 14.px, left 14.px。 默认尺寸为：宽为20.vp, 高为20.vp。

#### SwitchType

```cangjie
SwitchType
```
功能： 提供开关样式。 通用属性margin的默认值为：top 6.px, right 14.px, bottom 6.px, left 14.px。 默认尺寸为：宽为36.vp, 高为20.vp。

#### ButtonType

```cangjie
ButtonType
```
功能： 提供状态按钮样式，如果子组件有文本设置，则相应的文本内容会显示在按钮内部。 初始尺寸为：高为28.vp，宽无初始值。
