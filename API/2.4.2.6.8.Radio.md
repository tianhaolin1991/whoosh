# Radio

单选框，提供相应的用户交互选择项。

## Component Constructors

### public init(value!: String, group!: String, indicatorType!: RadioIndicatorType = RadioIndicatorType.TICK, indicatorBuilder!: Option<() -> Unit> = Option.None)

```cangjie
public init(value!: String, group!: String, indicatorType!: RadioIndicatorType = RadioIndicatorType.TICK, indicatorBuilder!: Option<() -> Unit> = Option.None)
```
功能： 创建单选框组件。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| String | 是 | 命名参数。 当前单选框的值。 | value | - |
| String | 是 | 命名参数。 当前单选框的所属群组名称，相同group的Radio只能有一个被选中。 | group | - |
| RadioIndicatorType | 否 | 命名参数。 配置单选框的选中样式。 | indicatorType | RadioIndicatorType.TICK |
| Option<()->Unit> | 否 | 命名参数。 配置单选框的选中样式为自定义UI描述。自定义UI描述与Radio组件为中心点对齐显示。indicatorBuilder设置为Option.None时，按照RadioIndicatorType.TICK进行显示。使用时结合 @Builder 和 bind 方法使用。 | indicatorBuilder | Option.None |

### public func radioStyle( checkedBackgroundColor!: ResourceColor = Color(0x007DFF), uncheckedBorderColor!: ResourceColor = Color(0x182431), indicatorColor!: ResourceColor = Color.WHITE ): This

```cangjie
public func radioStyle( checkedBackgroundColor!: ResourceColor = Color(0x007DFF), uncheckedBorderColor!: ResourceColor = Color(0x182431), indicatorColor!: ResourceColor = Color.WHITE ): This
```
功能： 设置单选框选中状态和非选中状态的样式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ResourceColor | 否 | 命名参数。 开启状态底板颜色。 | checkedBackgroundColor | Color(0x007DFF) |
| ResourceColor | 否 | 命名参数。 关闭状态描边颜色。 | uncheckedBorderColor | Color(0x182431) |
| ResourceColor | 否 | 命名参数。 开启状态内部圆饼颜色。indicatorType设置为RadioIndicatorType.TICK和RadioIndicatorType.DOT时，支持修改内部颜色。indicatorType设置为RadioIndicatorType.CUSTOM时，不支持修改内部颜色。 | indicatorColor | Color.WHITE |

## Enums

### RadioIndicatorType

enum RadioIndicatorType
```cangjie
public enum RadioIndicatorType { | TICK | DOT | CUSTOM }
```
功能： 单选框的选中样式。

#### CUSTOM

```cangjie
CUSTOM
```
功能： 选中样式为indicatorBuilder中的内容。

#### DOT

```cangjie
DOT
```
功能： 选中样式为系统默认DOT图标。

#### TICK

```cangjie
TICK
```
功能： 选中样式为系统默认TICK图标。
