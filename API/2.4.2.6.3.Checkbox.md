# Checkbox

多选框组件，通常用于某选项的打开或关闭。

## Component Constructors

### public init(name!: String = "", group!: String = "", indicatorBuilder!: ?()->Unit = None)

```cangjie
public init(name!: String = "", group!: String = "", indicatorBuilder!: ?()->Unit = None)
```
功能： 创建多选框组件。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| String | 否 | 命名参数。 多选框名称。 | name | "" |
| String | 否 | 命名参数。 用于指定多选框所属群组的名称（即所属 CheckboxGroup 的名称）。 说明 ： 未配合使用 CheckboxGroup 组件时，此值无用。 | group | "" |
| ?()->Unit | 否 | 命名参数。 配置多选框的选中样式为自定义UI描述。自定义UI描述与Checkbox组件为中心点对齐显示。indicatorBuilder设置为None时，默认为indicatorBuilder未设置状态。使用时结合 @Builder 和 bind 方法使用。 | indicatorBuilder | None |

## Examples

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 func build(){
 Flex(FlexParams(justifyContent: FlexAlign.SpaceAround, alignItems: ItemAlign.Center)){
 Checkbox(name: "checkbox1", group: "checkboxGroup")
 .select(true)
 .selectedColor(0xed6f21)
 .shape(CheckBoxShape.CIRCLE)
 .onChange({value: Bool =>
 BaseLog.info("Checkbox1 change is" + value.toString())
 })
 .size(width: 50.vp, height: 50.vp)
 Checkbox(name: "checkbox2", group: "checkboxGroup")
 .select(false)
 .selectedColor(0x39a2db)
 .shape(CheckBoxShape.ROUNDED_SQUARE)
 .onChange({value: Bool =>
 BaseLog.info("Checkbox2 change is" + value.toString())
 })
 .width(50.vp)
 .height(50.vp)
 }
 .height(400.vp)
 }
}
```

```cangjie
package ohos_app_cangjie_entry
import kit.UIKit.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
 @Builder
 func indicatorA() {
 Column() {
 Text("A").fontColor(Color.WHITE)
 }
 }
 func build() {
 Column() {
 Row() {
 Checkbox(name: "AAA", indicatorBuilder: bind(indicatorA, this))
 Text("AAA")
 }
 Row() {
 Checkbox(name: "BBB")
 Text("BBB")
 }
 Row() {
 Checkbox(name: "CCC").unselectedColor(Color.RED)
 Text("CCC")
 }
 Row() {
 Checkbox(name: "DDD").unselectedColor(Color.BLUE)
 Text("DDD")
 }
 }.width(100.percent)
 }
}
```
