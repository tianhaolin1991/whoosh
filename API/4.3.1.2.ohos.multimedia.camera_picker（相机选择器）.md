# ohos.multimedia.camera_picker（相机选择器）

本模块提供相机拍照与录制的能力。应用可以自行选择媒体类型实现拍照和录制的功能。该模块接口，需要应用在界面UIAbility中调用，否则无法拉起cameraPicker应用。

```cangjie
import kit.CameraKit.*
```
## Functions

### public func pick(context: UIAbilityContext, mediaTypes: Array<PickerMediaType>, pickerProfile: PickerProfile, callback: Callback1Argument<PickerResult>): Unit

```cangjie
public func pick(context: UIAbilityContext, mediaTypes: Array<PickerMediaType>, pickerProfile: PickerProfile, callback: Callback1Argument<PickerResult>): Unit
```
功能： 拉起相机选择器，根据媒体类型进入相应的模式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| UIAbilityContext | 是 | 应用上下文。 | context | - |
| Array< PickerMediaType > | 是 | 媒体类型。 | mediaTypes | - |
| PickerProfile | 是 | pickerProfile对象。 | pickerProfile | - |
| Callback1Argument < PickerResult > | 是 | 使用回调的方式获取相机选择器的处理结果。 | callback | - |
