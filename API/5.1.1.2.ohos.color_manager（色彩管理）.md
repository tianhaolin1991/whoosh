# ohos.color_manager（色彩管理）

本模块提供管理抽象化色域对象的一些基础能力，包括色域对象的创建与色域基础属性的获取等。

```cangjie
import kit.ArkGraphics2D.*
```
## Functions

### public func create(colorSpaceName: ColorSpace): ColorSpaceManager

```cangjie
public func create(colorSpaceName: ColorSpace): ColorSpaceManager
```
功能： 创建标准色域对象。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ColorSpace | 是 | 标准色域类型枚举值。UNKNOWN与CUSTOM不可用于直接创建色域对象。 | colorSpaceName | - |

**Returns:**

| type | description |
| --- | --- |
| ColorSpaceManager | 返回当前创建的色域对象实例。 |

**Example:**

```cangjie
// index.cj

import ohos.base.*
import kit.ArkGraphics2D.*

let colorSpaceManager = create(ColorSpace.SRGB)
```

### public func create(primaries: ColorSpacePrimaries, gamma: Float32): ColorSpaceManager

```cangjie
public func create(primaries: ColorSpacePrimaries, gamma: Float32): ColorSpaceManager
```
功能： 创建用户自定义色域对象。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| ColorSpacePrimaries | 是 | 色域标准三原色。 | primaries | - |
| Float32 | 是 | 色域gamma值。 | gamma | - |

**Returns:**

| type | description |
| --- | --- |
| ColorSpaceManager | 返回当前创建的色域对象实例。色域类型定义为 ColorSpace.CUSTOM 。 |

**Example:**

```cangjie
// index.cj

import ohos.base.*
import kit.ArkGraphics2D.*

let primaries = ColorSpacePrimaries(
 redX: 0.1,
 redY: 0.1,
 greenX: 0.2,
 greenY: 0.2,
 blueX: 0.3,
 blueY: 0.3,
 whitePointX: 0.4,
 whitePointY: 0.4
)
let gamma = 2.2f32
let colorSpaceManager = create(primaries, gamma)
```
