# ohos.multimedia_avsession（媒体会话管理）

媒体会话管理模块提供媒体播控相关功能的接口，目的是让应用接入播控中心。

```cangjie
import kit.AVSessionKit.*
```
## Functions

### public func createAVSession(context: CPointer<Unit>, tag: String, `type`: AVSessionType): AVSession

```cangjie
public func createAVSession(context: CPointer<Unit>, tag: String, `type`: AVSessionType): AVSession
```
功能： 创建会话对象，一个Ability只能存在一个会话，重复创建会失败。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| CPointer<Unit> | 是 | 需要使用 UIAbilityContext ，用于系统获取应用组件的相关信息。 | context | - |
| String | 是 | 会话的自定义名称。 | tag | - |
| AVSessionType | 是 | 会话的自定义名称。 | `type` | - |

**Returns:**

| type | description |
| --- | --- |
| AVSession | 回调返回会话实例对象，可用于获取会话ID，以及设置元数据、播放状态，发送按键事件等操作。 |

**Exceptions:**

- **BusinessException**: BusinessException：对应错误码的详细介绍请参见 媒体会话管理错误码 。

| code | description |
| --- | --- |
| 6600101 | Session service exception. |
