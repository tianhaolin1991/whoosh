# Row

沿水平方向布局的容器。

## Component Constructors

### public init(child: () -> Unit)

```cangjie
public init(child: () -> Unit)
```
功能： 创建一个包含子组件的Row容器。

### public init()

```cangjie
public init()
```
功能： 创建一个Row容器。

### public init(space: Length)

```cangjie
public init(space: Length)
```
功能： 创建一个横向布局元素间距为space的Row容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 横向布局元素间距。 space为负数或者justifyContent设置为FlexAlign.SpaceBetween、FlexAlign.SpaceAround、FlexAlign.SpaceEvenly时不生效。 初始值：0，单位：vp 说明： 可选值为大于等于0的数字。 | space | - |

### public init(space: Length, child: () -> Unit)

```cangjie
public init(space: Length, child: () -> Unit)
```
功能： 创建一个横向布局元素间距为space的Row容器。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| Length | 是 | 横向布局元素间距。 space为负数或者justifyContent设置为FlexAlign.SpaceBetween、FlexAlign.SpaceAround、FlexAlign.SpaceEvenly时不生效。 初始值：0，单位：vp 说明： 可选值为大于等于0的数字。 | space | - |
| ()->Unit | 是 | 容器内的子组件。 | child | - |


## Functions

### public func alignItems(algin: VerticalAlign): This

```cangjie
public func alignItems(algin: VerticalAlign): This
```
功能： 设置子组件在垂直方向上的对齐格式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| VerticalAlign | 是 | 在垂直方向上子组件的对齐格式。 初始值：VerticalAlign.Center | algin | - |

### public func justifyContent(algin: FlexAlign): This

```cangjie
public func justifyContent(algin: FlexAlign): This
```
功能： 设置子组件在水平方向上的对齐格式。

**Parameters:**

| type | required | description | name | default |
| --- | --- | --- | --- | --- |
| FlexAlign | 是 | 子组件在水平方向上的对齐格式。 初始值：FlexAlign.Start | algin | - |
